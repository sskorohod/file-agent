# FileAgent — AI File Intelligence Agent

## О проекте

FileAgent — персональный AI-агент для умного управления документами. Получает файлы через Telegram, веб-интерфейс или API, автоматически распознаёт, классифицирует, извлекает ключевую информацию и сохраняет в базу знаний с семантическим поиском.

**Ключевая идея:** загрузил документ — получил подробный анализ, рекомендации и напоминания. Задал вопрос — получил точный ответ на основе своих документов.

---

## Возможности

### 1. Умная обработка документов

- **Автоматическая классификация** — AI определяет тип документа (паспорт, анализ крови, контракт, счёт) и категорию (personal, business, health)
- **Извлечение полей** — из каждого документа извлекаются ключевые данные: ФИО, даты, суммы, диагнозы, номера документов
- **Подробный анализ** — AI объясняет что это за документ, для чего нужен, какие права даёт, что делать дальше
- **Рекомендации** — конкретные советы: когда продлить, куда обращаться, какие документы связаны, как хранить
- **Оценка важности** — приоритет (high/medium/low) с обоснованием

### 2. Многостраничное сканирование

- **Команда `/scan`** — пошаговое сканирование: отправляешь фото страниц одно за другим, система собирает в PDF
- **Media Group** — отправляешь несколько фото одним сообщением, автоматически собирается PDF
- **Auto-crop** — автоматическое обнаружение документа на фоне (стол, рука), обрезка рамок, выравнивание
- **Превью** — перед сборкой показывается полоска миниатюр для проверки порядка страниц

### 3. Семантический поиск (RAG)

- **Мультимодальные эмбеддинги** — Gemini Embedding 2 создаёт векторные представления текста И изображений
- **Qdrant** — быстрый векторный поиск по всем документам
- **LLM-синтез** — AI формирует подробный ответ на основе найденных документов
- **Контекст чата** — помнит историю разговора для уточняющих вопросов

### 4. Аналитика

- **Тренды** — "проанализируй гемоглобин за год" → график + текстовый анализ
- **Сравнения** — "сравни расходы по месяцам" → таблица + выводы
- **Графики** — автоматическая генерация PNG-графиков
- **Группировка данных** — извлечённые показатели группируются по метрике с нормами

### 5. AI Insights

- **Обзор по категориям** — AI анализирует все документы и даёт сводку состояния дел
- **Мотивирующие рекомендации** — не просто "продлите документ", а объяснение почему и как
- **Web Research** — опционально обогащает рекомендации актуальной информацией из интернета
- **Авто-обновление** — insights пересчитываются после каждого нового документа
- **Ежедневные советы** — утром в 9:00 и вечером в 20:00 приходит мотивирующий совет в Telegram

### 6. Напоминания

- **Автоматические** — при загрузке документа с датой окончания срока действия
- **Гибкие сроки** — паспорт: за 6 месяцев, водительские права: за 60 дней, остальные: за 2 недели
- **Telegram уведомления** — с кнопками "Готово" и "Отложить"
- **Дашборд** — виджет с ближайшими напоминаниями

### 7. Дедупликация

- **Точная** — по SHA-256 хешу (полное совпадение файлов)
- **Семантическая** — по vector similarity >94% (похожие документы)
- **Интерфейс выбора** — "оставить новый / старый / оба"

### 8. Голосовые сообщения

- **Транскрибация** — OpenAI Whisper распознаёт голос
- **Выбор действия** — поиск по документам или сохранение как заметка
- **Умные заметки** — LLM извлекает заголовок, теги, задачи, сохраняет как Obsidian .md

### 9. Веб-дашборд

- **Glassmorphism дизайн** — тёмная тема, Tailwind CSS, HTMX
- **KPI карточки** — файлы, векторы, запросы, LLM вызовы, напоминания, ошибки
- **Activity feed** — лента загрузок и поисковых запросов в реальном времени
- **Categories** — визуальное распределение по категориям
- **Pipeline health** — статус обработки файлов
- **Мобильная адаптация** — bottom navigation, touch-friendly, responsive

### 10. Система навыков (Skills)

- **YAML конфигурация** — каждый тип документа описывается в YAML файле
- **Извлечение полей** — настраиваемые поля для каждой категории
- **Шаблоны ответов** — форматирование вывода в Telegram
- **Горячая перезагрузка** — изменения в skills подхватываются каждые 30 секунд

## Поддерживаемые форматы

| Формат | Парсинг | Embedding |
|--------|---------|-----------|
| PDF | PyMuPDF + Tesseract OCR | Текст + мультимодальный |
| JPEG/PNG/HEIC | Vision API + Tesseract | Текст + мультимодальный |
| DOCX | python-docx | Только текст |
| TXT/CSV | Прямое чтение | Только текст |
| XLSX | openpyxl | Только текст |

## Технологии

- **Backend:** Python 3.14, FastAPI, uvicorn
- **Telegram:** python-telegram-bot (polling)
- **LLM:** litellm (Anthropic Claude, OpenAI GPT, Google Gemini)
- **Embedding:** Gemini Embedding 2 (768-dim multimodal)
- **Vector DB:** Qdrant (remote, SSH tunnel)
- **Metadata DB:** SQLite + aiosqlite (WAL mode, FTS5)
- **Parsers:** PyMuPDF, pytesseract, python-docx, OpenCV
- **Web UI:** Jinja2 + HTMX + Tailwind CSS CDN
- **Config:** YAML + Pydantic Settings + .env
